## ğŸ†˜ Troubleshooting

### Problem: Upload Failed
**Solution**: 
- Check file size (max 100MB)
- Verify file format is supported
- Try again - might be network issue

### Problem: Can't Find Photo
**Solution**:
- Check Trash folder first
- Search by partial filename
- Use advanced filters (date, location)
- Check album assignments

### Problem: Filters Stuck
**Solution**:
- Click **Reset** button in filter panel
- Clear search bar
- Refresh page (F5)

### Problem: Slideshow Stuck
**Solution**:
- Click next/previous arrows
- Exit and reopen slideshow
- Try different view mode

### Problem: Photo Won't Delete
**Solution**:
- Check if selected properly
- Deselect all, then select again
- Try single delete instead of batch
